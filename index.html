    Atlas

    Supporting the pillars of Agency data management for the Guatemala Mission.

    A data-informed, geographic approach to development

    Leverages the Google Workplace Tools You Know

    Atlas runs on Google Workplace—which Mission staff already know how to use. Plus, you only have to ask for data once. Atlas uses automated data pipelines to share data across applications.

    Integrates with Agency Systems

    Atlas helps to centralize data management across the Mission. Think of it as an extension of the Agency systems you already use for reporting and other data management tasks. It offers the functionality the Mission needs while easily integrating with existing Agency systems.

    Brings Data to Your Fingertips

    Atlas brings all of your activity-level data into the Mission Database for advanced queries. Answer questions and respond to taskers with ease.

    Make Data Discoverable with Custom Applications

    Atlas includes multiple out-of-the-box web applications. With all of your data in one place, the Mission's data champions can easily build custom applications using Google Apps Script, Tableau, or Esri ArcGIS.



    Custom web applications

    Atlas's interactive applications display and analyze data—helping you make connections to empower your work

    • Data Catalog: Search this catalog to find data to support evidence-based decision-making.
    • Map Viewer: See spatial distribution of key context indicators.
    • Activity Location Data Portal (Public): Share Implementing Partner Activity Location Data generalized for public consumption.

    Setup and maintenance

    Atlas is everyone at the Mission. But setting up and maintaining Atlas requires selecting a Data Steward.

    The Data Steward might be a Geographic Information Systems (GIS) Specialist, contractor through a Monitoring, Evaluation, and Learning (MEL) Platform, or just a data champion in the Mission. They'll need at least one day per week to support the Mission in the adoption and use of Atlas.

    The Mission is making progress towards meeting new ADS 579 requirements and onboarding to the Development Information System (DIS). The Data Steward will help develop the capacity and culture of the Mission to follow data best practices using Atlas. Long-term, this will pay dividends in Atlas's value for the Mission's data management.

    Data Stewards have a proficiency in database management and strong communication skills. Advanced users have experience with Google Apps Script. See the Atlas requirements page for more detail.


    Solutions not software

    Atlas is not software. Atlas is a data management approach for the Guatemala Mission. This site provides documentation for the Data Steward to implement and manage Atlas in the Guatemala Mission.

    If you or your Mission are looking for new ways to leverage activity-level data while using tools you're familiar with and integrated with Agency systems, consider Atlas. Atlas is free, open source and easy to set up. Click Get Started to begin. Contact erianderson@usaid.gov for more information.
